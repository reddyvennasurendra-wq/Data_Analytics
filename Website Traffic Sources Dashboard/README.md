# 📖 Project Overview

The **Website Traffic Sources Dashboard** is a Data Analytics project developed using **Python, Streamlit, Pandas, NumPy, and Matplotlib**. The dashboard analyzes website traffic from different sources and provides insights into visitor behavior using interactive charts and visualizations.

The project generates a sample dataset automatically, so no external CSV file is required.

---

# 🎯 Project Scope

The main objectives of this project are:

- Analyze website visitor traffic.
- Identify major traffic sources.
- Compare website visits across devices.
- Analyze browser usage.
- Study country-wise visitors.
- Monitor bounce rate and session duration.
- Generate business insights using interactive dashboards.

# ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Website-Traffic-Sources.git
```

Go to the project folder:

```bash
cd Website-Traffic-Sources
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
