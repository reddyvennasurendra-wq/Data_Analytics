# 📖 Project Description

The **Sales Trend Visualization Dashboard** is an interactive Data Analytics project developed using **Python, Pandas, Matplotlib, and Streamlit**. The dashboard analyzes sales data and provides meaningful insights through visualizations, KPI cards, and summary statistics.

The project demonstrates the complete Data Analytics workflow, including data loading, cleaning, analysis, visualization, and business insights.

---

# 🎯 Project Scope

This project helps businesses to:

- Analyze overall sales performance.
- Monitor sales trends.
- Identify high-performing categories.
- Compare sales across customer segments.
- Generate visual reports.
- Support business decision-making using data analytics.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Sales-Trend-Visualization.git
```

Move into the project directory

```bash
cd Sales-Trend-Visualization
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
