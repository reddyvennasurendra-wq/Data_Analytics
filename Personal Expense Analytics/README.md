# 📌 Project Overview

The **Personal Expense Analytics Dashboard** is a Data Analytics project developed using **Python**, **Pandas**, **Matplotlib**, and **Streamlit**. The project analyzes expense-related data to help users understand spending patterns, identify high-expense categories, and generate meaningful insights through interactive charts and dashboards.

The dashboard loads a dataset directly from a public URL, eliminating the need for manual CSV uploads.

---

# 🎯 Project Scope

The main objectives of this project are:

- Analyze personal expense data.
- Perform data cleaning and preprocessing.
- Generate statistical summaries.
- Visualize spending patterns.
- Identify highest expense categories.
- Create interactive dashboards.
- Generate business insights.
